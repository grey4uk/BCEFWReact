-----------------State---------------------------
- Class Components
- Внутрішній стан. початковий стан, що залежить від пропів
- Зміна стану
- Події: SyntheticEvent Object
- Асинхронність оновлення стану
- Зміна стану з використанням попереднього
- Викидування стану нагору
- Знаходження значення стану по ключу
- Тудушка: 
    - рендер списку
    - видалення
    - перемикання виконання завдання (чекбокс)
    - загальна кількість завдань і кількість виконаних

------------------Forms-----------------------------
- Неконтрольовані форми з event.currentTarget.elements
- Паттерн controlled element 
- Робота з input, checkbox, radio, select
- Форма реєстрації
- Генерація id для елементів форми
- Коллекція завдань: 
    - фінтрація по вмісту (текст)
